# Description
<img width="1017" height="149" alt="image" src="https://github.com/user-attachments/assets/37a98c58-5d8c-4b0d-ab02-165040dd70b2" />
**Wallfences** is a mod that, as the name suggests, allows walls and fences to connect. However, the mod doesn't stop there — it also adds new features, such as additional blocks in the Calcite palette and the Tinted Glass Pane. There are many more blocks and mechanics included, all focusing on improving completeness and enhancing how walls and fences behave and interact.
